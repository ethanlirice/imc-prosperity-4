# Analysis

This folder keeps the main research trail behind the strategies.

| Folder | Contents |
|---|---|
| `round3/` | Notebooks for product reverse engineering, Hydrogel behavior, option surfaces, and hidden-pattern discovery. |
| `round4/` | Informed-trader analysis, mark taxonomy, option validation, and Round 4 candidate ranking. |
| `round5/` | Consolidated Round 5 research by theme: group screens, hidden paths, mean reversion, market making, and pair-spread tests. |

The goal is to keep the analysis browsable while preserving the key evidence
behind the submitted strategy versions.

Manual challenge work lives separately in `manual-challenge/`, organized by
round.
