# Round 5 Analysis

Round 5 had the largest research surface, so the old experiment folders are
consolidated here by theme.

| Folder | Purpose |
|---|---|
| `initial-screen/` | Group tradeability, basket/spread screens, microstructure checks, and early residual simulations. |
| `hidden-paths/` | Hidden path and re-anchored fair-value research that became the main Round 5 breakthrough. |
| `free-alpha/` | Early hidden-path/free-alpha probes and backtest sweeps. |
| `mean-reversion/` | Product-local mean-reversion scans and executable probes. |
| `market-making-v7/` | Earlier market-making refinement tests. |
| `market-making-v11/` | Later market-making attribution, side gates, owned-product coexistence, and imbalance tests. |
| `mm-mr-framework/` | Combined market-making / mean-reversion framework prototype. |
| `pair-spreads/` | Pair-spread integration experiments. |
| `tls-pair-spreads/` | Rolling TLS pair-spread scans and passive/integrated sweeps. |
| `stat-arb-v11/` | Additional stat-arb checks around the v11 baseline. |
| `three-layer-prototype/` | Early three-layer strategy walk-forward and attribution work. |
| `owner-side/` | Owner-side ablation checks. |
| `global-identity-probes/` | Exploratory global identity / synchronization probes. |

The most important public findings are summarized in root `DATA.md` and
`TRADING.md`; this folder contains the scripts and supporting outputs.
