# Round 5 Analysis

Round 5 had the largest research surface. This folder keeps only the analysis
threads most relevant to the final strategy or to meaningful rejected paths.

| Folder | Purpose |
|---|---|
| `initial-screen/` | Early group, basket, lead-lag, and residual screens. Kept as the starting point for the Round 5 workflow. |
| `hidden-paths/` | Hidden path and re-anchored fair-value research that became the main Round 5 breakthrough. |
| `mean-reversion/` | Product-local mean-reversion scans and executable probes that informed promoted sleeves. |
| `market-making-v11/` | Market-making attribution, side gates, owned-product coexistence, and imbalance tests around the later baseline. |
| `neural-signals/` | NN/tree-assisted candidate generation and execution-aware integration summaries. |
| `tls-pair-spreads/` | Rolling TLS pair-spread scans and passive/integrated sweeps. |

The most important public findings are summarized in root `DATA.md` and
`TRADING.md`; this folder contains the scripts and compact supporting outputs.
