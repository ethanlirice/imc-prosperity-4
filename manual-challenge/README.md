# Manual Challenge

This folder keeps the non-algorithmic/manual challenge work, organized by
round.

| Folder | Contents |
|---|---|
| `round3/` | BioPods auction analysis notebook. |
| `round4/` | Exotic option pricing notes, optimizer, and selected output. |
| `round5/` | Ashflow Alpha news allocation optimizer and final recommendation. |

The goal is to keep the manual challenge evidence close to the final
recommendations without mixing it into the algorithmic strategy analysis.
